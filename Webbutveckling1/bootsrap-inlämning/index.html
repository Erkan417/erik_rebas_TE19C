Webbutvecklings begrepp

Element

Ett taggpar plus dess innehåll skapar tillsammans ett element. Se bild för att se hur ett element ser ut. Element har tre beståndsdelar först en starttagg sen innehåll och till sist en sluttagg notera att sluttagen även inkluderar ett /.

HTML

HTML är en förkortning för: HyperText Makeup Language. HTML är det vanligaste språket som webbsidor skrivs med.

ID

HTML- id-attributet används för att identifiera ett element med css i html dock kan det bara förekomma en id-attribut per html-sida.

Class

Klasser används för att strukturera olika objekt i en hemsida. Du kan även enkelt ha en egen css för varje klass.

Meta

Meta-taggen går alltid in i head-elementet. Med taggen kan du definiera nyckelord till sökmotoer, definiera hur ofta sida ska uppdateras och en beskrivning av hemsidan.

Responsive

Responsive handlar om att webbsidan ska justeras så det ser bättre ut, oberoende av enheten.

http

Hyper Text Transfer Protocol, mer känt som http. Ursprungliga syftet var att erbjuda en metod för HTML-sidor ut på informationsnätverket World Wide Web (www).

https

Hyper Text Transfer Protocol Secure (https), är den krypterade varianten av Hyper Text Transfer Protocol. Fördelen med https jämfört med http är att information är svårtillgänglig.

compressed data

Data som omkodas detta för att göra så att den tar mindre plats på datorn då färre informationsbärande enheter används.

CDN

CDN: Content Delivery Network, på svenska betyder det innehållsleveransnätverk. Huvudsyftet är att delge tjänster nära användaren genom att öka prestanda och driftsdäkerhet

local bootstrap

Bootstrap är ett bibliotek med färdigskriven css. Koden har även fördelen att kunna anpassa sig till skärmens dimensioner, då alla enheter inte har samma dimensioner.

JQuery

JQuery är likt bootstrap ett bibliotek för kod, skillnaden är att biblioteket är för JavaScript och inte till CSS som Bootstrap är.

Flexbox

Flexbox används för att strukturera sidan genom att säga var på sidan varje box med kod ska vara.

Grid

Grid är en rutnätslayout, fungerar som en mall för strukturen på sidan.

Code snippets

Code snippets är några rader färdigskriven kod som går att kopiera. Koden är ofta formellt definierade operativa enheter som ska införlivas i större programmeringsmoduler.
